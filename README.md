# Aura Brand

Boutique de moda feminina de luxo silencioso. Site editorial premium com React, Vite, Tailwind CSS e Framer Motion.

## Instalação

```bash
npm install
npm run dev
```

## Stack

- React 18
- Vite 5
- Tailwind CSS 3
- Framer Motion 11
- React Router DOM 6
- React Intersection Observer

## Estrutura de cores

| Cor              | Uso                        |
|------------------|----------------------------|
| Cream `#FAF7F2`  | Background principal       |
| Champagne `#C8A882` | Acento dourado          |
| Charcoal `#2C2926` | Texto e fundos escuros   |
| Blush `#E8D0C8`  | Acento feminino            |
